📋 QUICK SETUP (5 STEPS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Install dependencies:
   npm install
   cd web-panel && npm install && cd ..

2. Configure bot (.env):
   cp .env.example .env
   nano .env  # Add Discord token

3. Configure panel (web-panel/.env):
   cp web-panel/.env.example web-panel/.env
   nano web-panel/.env  # Add database password

4. Deploy commands:
   npm run deploy

5. Start everything:
   node index.js
